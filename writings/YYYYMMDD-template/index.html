To create a new blog post with this template:

  1. Copy this file to the directory of your new blog post.
  2. Update the marked tags in the <head> section at the top of the file and in the <section id="overview"> section above.
  3. Replace the tags in this area with your actual blog post content.
  4. Either remove the <aside> tag below, or fill it with your blog post's sidebar content.

Note: all areas of the template file that need to be updated are indicated with a starting HTML comment:

<!-- ↓↓↓ "..." ↓↓↓ -->

and an ending HTML comment:

<!-- ↑↑↑ "..." ↑↑↑ -->

For tags that only need to be updated, the text that requires updating is wrapped in brackets. For example:

<meta name="description" content="{Template Post Description}" />
